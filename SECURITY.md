# Security Policy

## Supported Versions

| Version | Supported |
| ------- | --------- |
| 1.2.x   | Yes       |
| < 1.0   | No        |

## Reporting a Vulnerability

If you discover a security vulnerability in ConnectionPool, please report it responsibly.

**Email**: security@olib.ai

Do **not** open a public GitHub issue for security vulnerabilities.

## What to Expect

- We will acknowledge your report within **48 hours**.
- We will work with you to understand and resolve the issue promptly.
- Once a fix is available, we will coordinate disclosure with you before publishing.

Thank you for helping keep ConnectionPool and its users safe.
